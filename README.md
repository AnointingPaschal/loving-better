# 💛 Loving Better

A mobile-first relationship growth tracker for couples ready to do the honest work of loving each other well.

## Features
- Daily, weekly, and monthly task checklists
- Automatic date-based resets (daily/weekly/monthly)
- Streak tracking that persists between sessions
- Reflection notes per section
- 7-day history calendar
- Progress rings and breakdowns
- Confetti celebrations on completion

## Tech
React + Vite + localStorage (no backend, no account needed)

## Run locally
```bash
npm install && npm run dev
```

## Deploy
Build with `npm run build` and deploy `dist/` to Netlify, Vercel, or GitHub Pages.
